# Mirrors the export-ignore patterns in each upstream plugin's
# .gitattributes — this is what wordpress.org's distribution builder
# drops on release. Keep the patterns generous: both plugins share
# most of these.
#
# DO NOT exclude:
#   - build/       (ActivityPub ships pre-built blocks here)
#   - composer.json / vendor/  (kept for forward-compat; Atmosphere
#                               currently uses a custom autoloader
#                               (includes/class-autoloader.php) and
#                               has no runtime composer deps, so the
#                               rsync'd vendor/ is just the composer
#                               autoload scaffolding. If upstream
#                               adds runtime deps later, sync-bundled
#                               already builds vendor/ via
#                               composer install --no-dev before rsync.)
#   - readme.txt   (WordPress.org plugin directory metadata)
#   - assets/, includes/, integration/, integrations/, patterns/,
#     templates/, languages/, *.php, LICENSE
